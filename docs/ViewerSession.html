Class: ViewerSession

ViewerSession

This class allows to manage state of selection and visibility in undo/redo session

Constructor

new ViewerSession(viewer)

This class is a convenience wrapper around the viewer which provides undo/redo operations for all viewer functions

Parameters:
Name Type Description
viewer Viewer

viewer to operate on
